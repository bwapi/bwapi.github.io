Function documentation

const UnitType::set& BWAPI::UnitTypes::allMacroTypes()

Retrieves the set of all macro unit types.

Returns A constant set of all macro unit types.

A macro type is a fake unit type, used by some functions. These include AllUnits, Men, Buildings, and Factories. The purpose of these types are to match the same ones used in Broodwar, also seen in the StarEdit map editor.

const UnitType::set& BWAPI::UnitTypes::allUnitTypes()

Retrieves the set of all defined unit types.

Returns A constant set of all available unit types.

int BWAPI::UnitTypes::maxUnitHeight()

Retrieves the maximum unit height from the set of all units.

Returns The maximum height of all unit types, in pixels.

Used internally to search through unit positions efficiently.

int BWAPI::UnitTypes::maxUnitWidth()

Retrieves the maximum unit width from the set of all units.

Returns The maximum width of all unit types, in pixels.

Used internally to search through unit positions efficiently.
